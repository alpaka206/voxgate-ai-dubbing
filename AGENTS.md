# AGENTS.md

## 프로젝트 목표

7일 인턴 과제를 위한 최소 기능의 AI 더빙 웹앱을 만든다.

## 과제 변경 반영

- 기존 과제는 텍스트를 입력해 음성을 생성하는 TTS 중심이었다.
- 변경된 과제는 오디오/비디오 파일을 업로드해 원하는 언어로 더빙하는 파일 기반 파이프라인이 핵심이다.
- 따라서 본 서비스는 `파일 업로드 -> 음성 추출 -> 전사 -> 번역 -> 타깃 언어 음성 합성 -> 재생/다운로드` 흐름을 제공해야 한다.

## 제품 범위

- Next.js App Router
- Google OAuth 로그인
- Turso allowlist 기반 접근 제어
- 시드 데이터에 `kts123@estsoft.com` 포함
- 로그아웃 상태 사용자는 랜딩 페이지 노출
- 미승인 사용자는 접근 제한 안내 노출
- self-approval 없이 허용 목록에 등록된 이메일만 스튜디오 이용 가능
- 미승인 사용자는 허용 신청 가능
- `kts123@estsoft.com`, `gyuwon05@gmail.com`는 `manager` 역할로 허용 요청 승인 및 member 추가 가능
- 승인된 사용자는 오디오/비디오 파일 업로드, 타깃 언어 선택, 더빙 생성, 재생/다운로드 가능
- Vercel 배포
- `README.md` 문서화

## 작업 원칙

- 구현은 단순하고 리뷰 가능하게 유지한다.
- 불필요한 관리자 기능은 만들지 않는다.
- 안정적인 라이브러리와 최소 의존성을 우선한다.
- 코딩 전에 짧은 계획을 먼저 제안한다.
- 변경은 작은 단계로 나눈다.
- 동작이 바뀌면 `README.md`, `.env.example`, `docs/agent-log.md`를 함께 업데이트한다.
- 의미 있는 수정 후에는 lint/build를 실행한다.
